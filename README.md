# 👟 Nike - Landing Page

Projeto de uma landing page da **Nike**, desenvolvido com **HTML, CSS e JavaScript**.  
O site apresenta um tênis da linha **Nike Air Zoom**, com troca dinâmica de cores e animações suaves.

---

## 🚀 Tecnologias utilizadas
- **HTML5**
- **CSS3**
- **JavaScript**
